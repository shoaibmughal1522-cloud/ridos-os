🖥 Overview

RIDOS OS is a bootable Linux distribution built for IT professionals — network engineers, system administrators, security researchers, and developers. It combines a full suite of professional tools with a conversational AI assistant powered by Anthropic Claude.

Boot from a USB stick on any x86_64 machine — old or new — no installation required.

Debian Bookworm Base Claude AI Integration BIOS + UEFI Boot Dark Purple XFCE Desktop GPL v3 Open Source Works on 512MB RAM

What makes RIDOS different?

Unlike Kali Linux (focused on penetration testing) or Ubuntu (general purpose), RIDOS integrates AI into every IT task. Instead of remembering commands, you ask RIDOS:

RIDOS AI Shell — examples
You:   scan my local network
RIDOS: Running nmap -sn 192.168.1.0/24...
       Found 8 hosts: 192.168.1.1 (router), 192.168.1.5 (your PC)...

You:   what ports is my server running?
RIDOS: Running ss -tlnp...
       Port 22 (SSH), Port 80 (HTTP), Port 3306 (MySQL)

You:   CVE-2024-1234
RIDOS: [Web Search] CVE-2024-1234 — Critical RCE in OpenSSH...
       Severity: 9.8 CRITICAL | Fix: Update to OpenSSH 9.7+

💾 Installation — Flash to USB

Download the ISO

Download the latest ISO from the GitHub Releases page:

URL
https://github.com/ridos-os-project/ridos-os/releases

Flash on Windows

Use Balena Etcher — free, works with any ISO.

Steps
1. Download Balena Etcher from etcher.balena.io
2. Insert USB drive (2GB minimum — ALL DATA WILL BE ERASED)
3. Open Etcher → Flash from file → select ridos-os-1.1.0-x86_64.iso
4. Select your USB drive
5. Click Flash — wait ~5 minutes
6. Done — safely eject USB

Flash on Linux / Mac

bash
# Find your USB drive letter
lsblk

# Flash (replace /dev/sdX with your USB drive)
sudo dd if=ridos-os-1.1.0-x86_64.iso \
        of=/dev/sdX \
        bs=4M status=progress conv=fsync

# Verify (optional but recommended)
md5sum -c ridos-os-1.1.0-x86_64.iso.md5

Boot from USB

Boot key by manufacturer
Dell      →  F12
HP        →  F9  or  Esc
Lenovo    →  F12  or  F1

Acer      →  F12

⚠ UEFI Secure Boot If your PC refuses to boot RIDOS, go to BIOS settings and disable Secure Boot. RIDOS does not have a Microsoft-signed bootloader.

🚀 First Boot

On first boot RIDOS runs the Setup Wizard automatically. It takes 2–3 minutes and configures everything.

StepWhat happens
1 — API KeyEnter your Anthropic Claude API key to enable AI features
2 — NetworkTests internet connectivity to Google, Cloudflare, and Anthropic
3 — ProfileYour name and timezone
4 — ThemeChoose shell color (default: dark purple)
5 — SSH KeysGenerates Ed25519 SSH key pair for remote access
6 — SecurityChecks firewall, SSH config, password status

🤖 RIDOS AI Shell

The RIDOS AI Shell is your main interface. Launch it from the desktop or terminal:

bash
ridos-shell
# or
python3 /opt/ridos/bin/ridos_shell.py

How it works

Type anything in plain language. RIDOS decides whether to run a system command, search the web, or ask Claude AI — and returns a clear answer.

Conversation flow

# Copy file from server
scp user@192.168.1.100:/etc/nginx/nginx.conf ./

# SSH tunnel (forward local port 8080 to remote port 80)
ssh -L 8080:localhost:80 user@192.168.1.100

# Run command on remote server

🔨 Build from Source

bash — Full build on Ubuntu 22.04
# 1. Clone the repo
git clone https://github.com/ridos-os-project/ridos-os.git
cd ridos-os

# 2. Install build dependencies
sudo apt install live-build debootstrap squashfs-tools xorriso

# 3. Configure live-build
cd build-system
sudo bash scripts/lb_config.sh

# 4. Build the ISO (~30-60 minutes)
sudo lb build

# Output: live-image-amd64.hybrid.iso (~800MB)
✓ No Linux machine? Use GitHub Actions Push to GitHub → Actions tab → Run workflow → Download ISO from Artifacts. Completely free for public repos.
